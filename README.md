This repository contains the raw functions for clustering the scRNAseq data (***RunSeuratV5.R***) and identifying differentially expressed genes (***FindMarkersSCTV5.R***), clustering the RNAscope data (***ClusterRNAscopeData.R***), and spatial registration functions for building an atlas of the claustro-insular region (***spatial_registration_helper_functions.R***). The complete codes for reproducing all graphs generated with R in the manuscript will be made available alongside the raw data upon publication of the manuscript.
